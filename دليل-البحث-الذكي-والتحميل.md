# البحث الذكي وتحميل الدراسات - دليل سريع

## 🎯 الميزات الجديدة

تم إضافة خاصية **البحث الذكي** و**التحميل المتطور** لقسم الدراسات عن الإمام الونشريسي في المكتبة الرقمية.

---

## 🔍 البحث الذكي

### كيفية الاستخدام
1. افتح المكتبة الرقمية (library.html)
2. انتقل إلى قسم "الدراسات والأبحاث عن الإمام الونشريسي"
3. استخدم صندوق البحث الكبير

### مزايا البحث
✅ **بحث شامل** في جميع حقول الدراسة:
- العنوان
- المؤلف
- الوصف
- جهة النشر
- التصنيف
- نوع الدراسة

✅ **نتائج فورية** أثناء الكتابة (بعد 300 ميللي ثانية)

✅ **تمييز بصري** للكلمات المطابقة بخلفية صفراء

✅ **عداد النتائج** يظهر عدد الدراسات المطابقة

✅ **زر مسح سريع** (❌) لإعادة عرض جميع الدراسات

✅ **دمج البحث والفلترة** - ابحث ضمن نوع محدد

---

## 💡 أمثلة على البحث

```
"فقه"          → كل الدراسات المتعلقة بالفقه
"الغرياني"     → دراسات المؤلف الغرياني
"2021"         → دراسات سنة 2021
"دكتوراه"      → أطروحات الدكتوراه فقط
"منهج المعيار" → دراسات عن منهج المعيار
"تلمسان"       → دراسات من جامعة تلمسان
```

---

## 📥 تحميل الدراسات

### للدراسات المتوفرة (hasFile: true)
1. **زر "عرض"** 👁️ - يفتح الدراسة في نافذة جديدة
2. **زر "تحميل"** ⬇️ - يحمل ملف PDF مباشرة
   - يظهر إشعار بصري عند بدء التحميل
   - يحفظ سجل التحميلات في المتصفح

### للدراسات غير المتوفرة (hasFile: false)
1. **زر "طلب النسخة"** 📧
2. أدخل بريدك الإلكتروني
3. يتم حفظ الطلب
4. سيتم إرسال الدراسة عند توفرها

- 🟡 **تمييز النص المطابق** - خلفية صفراء

## 🛠️ للمطورين

### الملفات المعدلة
1. **library.html** - إضافة صندوق البحث و CSS
2. **library.js** - وظائف البحث والتحميل
3. **مجلد جديد**: `الدراسات عن الونشريسي/`

### الوظائف الرئيسية

```javascript
// البحث
initSearchBox()              // تهيئة البحث
applyFiltersAndSearch()      // تطبيق البحث والفلترة
highlightSearchTerms(card)   // تمييز الكلمات

// التحميل
downloadStudy(id)            // تحميل دراسة
requestStudy(id)             // طلب دراسة
showDownloadNotification()   // إظهار إشعار
```

### متغيرات البحث
```javascript
currentSearchTerm = ''  // كلمة البحث الحالية
currentFilter = 'all'   // الفلتر الحالي
searchTimeout = null    // للتأخير الذكي
```

## 🔄 سير العمل

### البحث:
```
المستخدم يكتب → تأخير 300ms → تطبيق البحث → تمييز النتائج → عرض العدد
```

### التحميل:
```
ضغط زر التحميل → إنشاء رابط → حفظ في السجل → تحميل الملف → إظهار إشعار
```

### طلب دراسة:
```
ضغط زر الطلب → إدخال البريد → التحقق → حفظ الطلب → تأكيد
```

## 🎯 نصائح الاستخدام

### للحصول على أفضل النتائج:
1. **استخدم كلمات مفتاحية واضحة**
2. **جرب البحث بكلمات مختلفة** (عربي/إنجليزي)
3. **استخدم الفلترة مع البحث** لتضييق النتائج
4. **امسح البحث** للعودة لجميع الدراسات

### لإضافة دراسات جديدة:
1. ضع ملف PDF في مجلد `الدراسات عن الونشريسي/`
2. أضف بيانات الدراسة في `library.js`
3. اضبط `hasFile: true` و `filename: "اسم_الملف.pdf"`

---

## ⚡ الأداء

- **البحث الفوري**: نتائج في أقل من ثانية
- **تأخير ذكي**: 300ms لتجنب البحث المتكرر
- **تمييز آمن**: استخدام regex آمن
- **حفظ محلي**: لا حاجة لخادم

## 🆘 حل المشاكل

### البحث لا يعمل؟
- تحقق من وجود `id="studiesSearchInput"`

- تأكد من وجود الملف في المجلد الصحيح
- تحقق من تطابق اسم الملف

### النتائج غير دقيقة؟
- البحث يبحث في جميع الحقول
- استخدم كلمات أكثر دقة
