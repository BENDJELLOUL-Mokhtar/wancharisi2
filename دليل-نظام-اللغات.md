# 🌐 دليل نظام اللغات الثنائي (عربي - إنجليزي)

## ✨ نظرة عامة

تم إضافة نظام ترجمة ثنائي اللغة شامل للمنصة بأكملها، يدعم **العربية** و**الإنجليزية** مع إمكانية التبديل السلس بينهما.

### 1. **تبديل اللغة الفوري**

- تغيير فوري بدون إعادة تحميل

- العناوين والأقسام

## 🔧 للمطورين

### إضافة ترجمة لعنصر جديد:

#### 1. **إضافة المفتاح في translations.js**
```javascript
const translations = {
    ar: {
        new_key: "النص بالعربية",
        // ...
    },
    en: {
        new_key: "Text in English",
        // ...
    }
};
```

#### 2. **ربط العنصر بالترجمة**
```html
<!-- للنصوص العادية -->
<h1 data-i18n="new_key">النص بالعربية</h1>

<!-- للـ placeholders -->
<input data-i18n-placeholder="new_key" placeholder="النص بالعربية">

<!-- للـ title attributes -->
<button data-i18n-title="new_key" title="النص بالعربية">...</button>
```

#### 3. **استخدام دالة الترجمة في JavaScript**
```javascript
// الحصول على نص مترجم
const text = languageManager.translate('new_key');

// أو استخدام الدالة المختصرة
const text = t('new_key');
```

## 🔄 آلية العمل

### 1. **التهيئة الأولية**
```
1. تحميل الصفحة
2. تشغيل translations.js
3. إنشاء LanguageManager
4. قراءة اللغة المحفوظة (أو العربية افتراضياً)
5. تطبيق الترجمة على جميع العناصر
```

### 2. **عند التبديل**
```
1. المستخدم يضغط على زر اللغة
2. toggleLanguage() تُستدعى
3. تغيير currentLang
4. حفظ في localStorage
5. تحديث dir و lang في <html>
6. تطبيق الترجمة الجديدة
7. تحديث نص الزر
```

### 3. **الحفظ والاسترجاع**
```javascript
// الحفظ
localStorage.setItem('wancharisi_language', 'ar' أو 'en');

// الاسترجاع
const lang = localStorage.getItem('wancharisi_language') || 'ar';
```

## 🐛 استكشاف الأخطاء

### المشكلة: الترجمة لا تظهر
**الحل**:
```
1. تحقق من data-i18n في HTML
2. تحقق من وجود المفتاح في translations.js
3. افتح Console (F12) لرؤية الأخطاء
4. تأكد من تحميل translations.js قبل باقي السكربتات
```

### المشكلة: اللغة لا تُحفظ
**الحل**:
```
1. تحقق من localStorage في المتصفح
2. تأكد من عدم استخدام وضع التصفح الخاص
3. امسح localStorage وجرب مرة أخرى
```

### المشكلة: الاتجاه لا يتغير (RTL/LTR)
**الحل**:
```
1. تحقق من تحديث dir في <html>
2. تحقق من أنماط CSS لـ html[dir="ltr"]
3. امسح الكاش وأعد تحميل الصفحة
```

### المشكلة: زر اللغة لا يعمل
**الحل**:
```
1. تحقق من id="languageToggle" في HTML
2. تحقق من attachEventListeners() في JS
3. تحقق من Console للأخطاء
```

## 🎓 أمثلة عملية

### مثال 1: ترجمة عنوان
```html
<!-- قبل -->
<h1>المكتبة الرقمية</h1>

<!-- بعد -->
<h1 data-i18n="library_title">المكتبة الرقمية</h1>
```

### مثال 2: ترجمة زر
```html
<!-- قبل -->
<button>حفظ</button>

<!-- بعد -->
<button data-i18n="save">حفظ</button>
```

### مثال 3: ترجمة placeholder
```html
<!-- قبل -->
<input placeholder="ابحث...">

<!-- بعد -->
<input data-i18n-placeholder="search_placeholder" placeholder="ابحث...">
```

### مثال 4: استخدام في JavaScript
```javascript
// عرض رسالة مترجمة
function showMessage() {
    const message = t('notification_success');
    alert(message);  // "تم بنجاح!" أو "Success!"
}

// تحديد اللغة الحالية
const currentLang = languageManager.getCurrentLanguage();
if (currentLang === 'ar') {
    // منطق خاص بالعربية
} else {
    // منطق خاص بالإنجليزية
}
```
