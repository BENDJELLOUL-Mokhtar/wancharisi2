# دليل قاعدة بيانات النوازل التاريخية والرقمية

## 📚 نظرة عامة

قاعدة بيانات شاملة تربط بين النوازل الفقهية التي عالجها الإمام الونشريسي في عصره والنوازل الرقمية المعاصرة، وفق منهجية علمية دقيقة.

---

## 🎯 الهدف من القاعدة

1. **الربط التاريخي:** إيجاد علاقة منهجية بين النوازل التراثية والمعاصرة
2. **التحليل الفقهي:** تقديم أحكام فقهية مستندة إلى منهج الونشريسي
3. **التطبيق العملي:** توفير أمثلة واقعية للنوازل الرقمية
4. **التعليم والتدريب:** مساعدة الباحثين على فهم منهج فقه النوازل

---

## 🔍 هيكل البيانات

### كل نازلة تحتوي على:

#### 1. **الحالة التاريخية** (historicalCase)
```javascript
{
    title: "عنوان النازلة التاريخية",
    description: "وصف تفصيلي للمسألة كما عالجها الونشريسي",
    reference: "المرجع الدقيق من مؤلفات الونشريسي",
    ruling: "الحكم الفقهي المستخرج"
}
```

#### 2. **الحالة الرقمية** (digitalCase)
```javascript
{
    title: "عنوان النازلة الرقمية المعاصرة",
    description: "شرح الحالة الرقمية المعاصرة",
    examples: ["مثال 1", "مثال 2", "مثال 3"]
}
```

#### 3. **تحليل المطابقة** (matching)
```javascript
{
    type: "نوع المطابقة",  // مطابقة دقيقة، متشابهة، بالقياس، بالقواعد
    similarity: 95,  // نسبة التطابق من 0-100
    reasoning: "تفسير وجه المطابقة",
    conditions: ["شرط 1", "شرط 2", "شرط 3"]
}
```

## 💡 أمثلة تطبيقية

### مثال 1: عرض نازلة مع التحليل
```javascript
const nazila = NawazilDatabase.getNazila(1);

console.log("النازلة الرقمية:", nazila.digitalCase.title);
console.log("النازلة التاريخية:", nazila.historicalCase.title);
console.log("نسبة التطابق:", nazila.matching.similarity + "%");
console.log("الحكم:", nazila.historicalCase.ruling);
```

### مثال 2: البحث عن نوازل عالية التطابق
```javascript
const highQualityMatches = NawazilDatabase.getBysimilarity(90);

highQualityMatches.forEach(nazila => {
    console.log(`✅ ${nazila.digitalCase.title} - ${nazila.matching.similarity}%`);
});
```

### مثال 3: عرض أمثلة عشوائية
```javascript
const examples = NawazilDatabase.getRandom(3);

examples.forEach((nazila, index) => {
    console.log(`${index + 1}. ${nazila.digitalCase.title}`);
    console.log(`   المرجع: ${nazila.historicalCase.reference}`);
});
```

## 🔬 المنهجية العلمية

### معايير اختيار النوازل:

1. **الأصالة:** نوازل حقيقية من مؤلفات الونشريسي
2. **الواقعية:** مسائل رقمية معاصرة فعلية
3. **التنوع:** تغطية مختلف المجالات الرقمية
4. **الدقة:** تحليل فقهي دقيق للمطابقة

### أنواع المطابقة:

1. **مطابقة دقيقة:** تطابق في الماهية والأحكام
2. **مطابقة متشابهة:** تشابه في الطبيعة مع بعض الفروق
3. **مطابقة بالقياس:** قياس فقهي على الأصل
4. **مطابقة بالقواعد:** تطبيق القواعد الفقهية العامة
