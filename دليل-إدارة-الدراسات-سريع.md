# ⚡ دليل سريع - إدارة الدراسات

## 🎯 في 30 ثانية

### ✏️ **تعديل دراسة**:
```
1. افتح library.html
2. ابحث عن دراسة مرفوعة
3. اضغط زر "✏️ تعديل" (أزرق)
4. عدّل ما تريد
5. اضغط "💾 حفظ"
```

### 🗑️ **حذف دراسة**:
```
1. افتح library.html
2. ابحث عن دراسة مرفوعة
3. اضغط زر "🗑️ حذف" (أحمر)
4. وافق على الحذف
5. ✅ تم!
```

---

## 🔍 أين الأزرار؟

### دراسة مرفوعة:
```
[👁️ عرض] [⬇️ تحميل]
[✏️ تعديل] [🗑️ حذف] ← هنا!
```

### دراسة تجريبية:
```
[👁️ عرض] [⬇️ تحميل]
(لا توجد أزرار إدارة) ← محمية
```

| الحقل | مثال |
|-------|------|
| 📚 النوع | مقال، كتاب، رسالة... |
| 📖 العنوان | "دراسة فقهية..." |
| ✍️ المؤلف | "د. أحمد..." |
| 📅 السنة | 2025 |
| 📄 الصفحات | 120 |
| 🏢 الناشر | "دار النشر..." |

| 📝 الوصف | نص كامل |
| 🌐 اللغة | عربية، فرنسية... |
| 🔑 الكلمات | مفصولة بفواصل |
| 🔗 الرابط | https://... |

## ❌ ما لا يمكن تعديله

- 📄 **الملف نفسه** (PDF) - احذف وأعد رفعه
- 🆔 **المعرف** (ID)
- 📅 **تاريخ الرفع**
- ✅ **حالة الموافقة**

---

## ⚠️ تحذيرات

### الحذف:
```
⚠️ نهائي - لا يمكن التراجع!
⚠️ يُحذف الملف والبيانات معاً
⚠️ لا توجد سلة مهملات (حالياً)
```

### التعديل:
```
⚠️ املأ الحقول المطلوبة (*)
⚠️ تُحفظ فوراً في localStorage
⚠️ تتحديث تلقائياً في الصفحة
```

- ✅ يُحفظ في localStorage
- ✅ متاح بعد إعادة التحميل

### بعد الحذف:
- ✅ يختفي فوراً
- ✅ يُحذف نهائياً
- ✅ لا يعود بعد إعادة التحميل
