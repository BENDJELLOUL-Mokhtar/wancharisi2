# دليل قسم الدراسات والأبحاث عن الإمام الونشريسي

## نظرة عامة

تم إضافة قسم جديد في المكتبة الرقمية مخصص للدراسات والأبحاث التي تناولت سيرة وتراث الإمام أحمد بن يحيى الونشريسي، ويشمل:
- **الكتب** الأكاديمية والبحثية
- **المقالات** العلمية المحكمة
- **أطروحات الدكتوراه** والماجستير
- **أعمال الملتقيات والمؤتمرات** العلمية

---

## المزايا الرئيسية

### 1. التصنيف حسب نوع الدراسة
- **كتب**: دراسات شاملة ومعمقة
- **مقالات**: أبحاث علمية محكمة
- **أطروحات**: رسائل جامعية (ماجستير ودكتوراه)
- **ملتقيات**: أعمال المؤتمرات والندوات العلمية

### 2. نظام الفلترة
- إمكانية تصفية الدراسات حسب النوع
- عرض جميع الدراسات أو فلترتها حسب الفئة
- واجهة سهلة الاستخدام مع أزرار تصفية واضحة

### 3. البحث الذكي ⭐ جديد
- **بحث شامل**: في العنوان، المؤلف، الوصف، جهة النشر، والتصنيف
- **بحث فوري**: نتائج فورية أثناء الكتابة
- **تمييز النتائج**: تظهر الكلمات المطابقة بلون بارز
- **عداد النتائج**: يظهر عدد الدراسات المطابقة
- **مسح سريع**: زر لمسح البحث والعودة لجميع الدراسات
- **دمج البحث والفلترة**: إمكانية البحث ضمن فئة معينة

### 4. معلومات تفصيلية
كل دراسة تحتوي على:
- **العنوان الكامل**
- **اسم المؤلف/الباحث**
- **نوع الدراسة**
- **سنة النشر**
- **عدد الصفحات**
- **وصف مختصر للمحتوى**
- **جهة النشر أو الجامعة**
- **التصنيف الموضوعي**

### 5. خيارات متعددة
- **عرض**: لعرض الدراسة (للدراسات المتوفرة رقمياً)
- **تحميل**: لتحميل نسخة PDF مع إشعار فوري
- **طلب النسخة**: للدراسات غير المتوفرة حالياً (مع إدخال البريد الإلكتروني)

### 6. إحصائيات مباشرة
- عدد الكتب المتوفرة
- عدد المقالات
- عدد الأطروحات
- عدد الملتقيات والمؤتمرات

### 7. تتبع التحميلات والطلبات
- حفظ سجل التحميلات في المتصفح
- حفظ طلبات الدراسات غير المتوفرة
- إشعارات بصرية عند التحميل

## كيفية استخدام البحث الذكي

### البحث البسيط
1. اكتب أي كلمة في صندوق البحث
2. ستظهر النتائج فوراً أثناء الكتابة
3. الدراسات المطابقة تظهر مع تمييز الكلمات المطابقة بلون أصفر

### البحث المتقدم
يمكنك البحث في:
- **العنوان**: مثل "المعيار" أو "النوازل"
- **المؤلف**: مثل "الغرياني" أو "العلمي"
- **الوصف**: أي كلمة من وصف الدراسة
- **جهة النشر**: مثل "جامعة" أو "دار الكتب"
- **التصنيف**: مثل "فقهية" أو "تاريخية"

### دمج البحث والفلترة
1. اختر نوع الدراسة من أزرار الفلترة (كتب، مقالات، إلخ)
2. ثم اكتب في صندوق البحث
3. ستظهر فقط النتائج التي تطابق النوع والبحث معاً

### مسح البحث
- اضغط على زر ❌ في صندوق البحث
- أو امسح النص يدوياً

### أمثلة على البحث
```
"فقه" → يظهر كل الدراسات المتعلقة بالفقه
"2021" → يظهر الدراسات المنشورة في 2021
"دكتوراه" → يظهر أطروحات الدكتوراه
"تلمسان" → يظهر الدراسات من جامعة تلمسان
"منهج" → يظهر الدراسات التي تتناول المنهج
```

## كيفية تحميل الدراسات

### للدراسات المتوفرة
1. اضغط على زر **"عرض"** لمشاهدة الدراسة في نافذة جديدة
2. أو اضغط على زر **"تحميل"** لتنزيل ملف PDF
3. سيظهر إشعار في أسفل الصفحة عند بدء التحميل

### للدراسات غير المتوفرة
1. اضغط على زر **"طلب النسخة"**
2. أدخل بريدك الإلكتروني
3. سيتم حفظ طلبك وإرسال الدراسة عند توفرها

## كيفية إضافة دراسات جديدة

### الخطوة 1: فتح ملف library.js
ابحث عن المصفوفة `studies` (حوالي السطر 70-150)

### الخطوة 2: إضافة عنصر جديد
```javascript
{
    id: 7, // رقم تسلسلي جديد
    title: 'عنوان الدراسة',
    author: 'اسم المؤلف',
    type: 'نوع الدراسة', // كتاب، مقال، أطروحة دكتوراه، ملتقى
    year: 'سنة النشر',
    pages: 'عدد الصفحات',
    description: 'وصف مختصر للدراسة ومحتواها',
    publisher: 'جهة النشر',
    category: 'التصنيف الموضوعي',
    filename: 'اسم الملف.pdf', // أو '' إذا لم يكن متوفراً
    hasFile: true // true إذا كان الملف متوفراً، false إذا لم يكن
}
```

### الخطوة 3: تحديث الإحصائيات
في ملف library.html، قم بتحديث الأرقام في قسم `studies-stats`:
```html
<div class="stat-box">
    <i class="fas fa-book"></i>
    <span class="stat-num">العدد الجديد</span>
    <span class="stat-label">كتاب</span>
</div>
```

### الخطوة 4: رفع ملف PDF (اختياري)
إذا كان لديك نسخة PDF من الدراسة:
1. أنشئ مجلد جديد: `الدراسات عن الونشريسي/`
2. ضع ملفات PDF فيه
3. حدّث `filename` في مصفوفة الدراسات
4. اضبط `hasFile: true`

## الدراسات النموذجية المضافة

تم إضافة 6 دراسات نموذجية:

1. **الإمام الونشريسي وجهوده الفقهية** (كتاب)
   - دراسة شاملة عن حياته ومنهجه الفقهي

2. **منهج الونشريسي في المعيار المعرب** (مقال)
   - تحليل منهج جمع الفتاوى

3. **النوازل الفقهية عند الونشريسي وأثرها المعاصر** (أطروحة دكتوراه)
   - دراسة أكاديمية تحليلية

4. **الونشريسي والحياة الاجتماعية في المغرب الإسلامي** (كتاب)
   - دراسة تاريخية اجتماعية

5. **القواعد الفقهية في كتاب إيضاح المسالك** (مقال)
   - تحليل القواعد الفقهية

6. **أعمال الملتقى الدولي حول الإمام الونشريسي** (ملتقى)
   - مجموع أعمال المؤتمر

> **ملاحظة**: هذه دراسات نموذجية لأغراض التوضيح. يمكن استبدالها بدراسات حقيقية.
