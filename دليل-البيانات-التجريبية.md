### 3️⃣ محرك الأفكار الذكي (ai-engine.html)

#### المحادثة التجريبية:

**👤 المستخدم:**
> ما حكم استخدام منصات التواصل الاجتماعي في نشر الفتاوى الشرعية؟

**🤖 مساعد الذكاء الاصطناعي:**
> بناءً على منهج الإمام الونشريسي في التعامل مع النوازل، يمكن تقسيم الحكم إلى عدة جوانب:
> 
> **1. الأصل في وسائل النشر:**
> وسائل النشر الحديثة تُلحق بباب الوسائل لا المقاصد. والأصل في الوسائل الإباحة ما لم تؤدِ إلى محظور.
> 
> **2. الشروط الواجبة:**
> - التأكد من صحة المعلومة قبل النشر
> - عدم الفتوى بغير علم
> - مراعاة السياق والظروف
> 
> **المراجع:**
> - المعيار المعرب، الجزء 6، ص89-102
> - إيضاح المسالك، الجزء 2، ص234-256

**👤 المستخدم:**
> هل يوجد نص صريح في هذا الموضوع؟

**🤖 مساعد الذكاء الاصطناعي:**
> لا يوجد نص صريح في المنصات الرقمية تحديداً، لكن يمكن القياس على أحكام "نشر العلم" و"التبليغ" في الفقه المالكي.
> 
> **القياس المقترح:**
> منصات التواصل ← نشر العلم في المجالس العامة
> 
> **المراجع:**
> - المنهج الفائق، المجلد 1، ص456-478

**👨‍🏫 خبير فقهي:**
> ✅ تعليق محقق: 
> ينبغي التفريق بين النشر العام والخاص، فالفتوى العامة تحتاج إلى مزيد من التحقق والتثبت.
> 
> **إضافة مهمة:**
> - النوازل الجامعة، المجلد 3، ص123-145

## ⚙️ كيفية عمل النظام

### التحميل التلقائي:
```javascript
// عند زيارة المنصة لأول مرة
if (!localStorage.getItem('demo_data_loaded')) {
    DemoData.loadAllDemoData();
    localStorage.setItem('demo_data_loaded', 'true');
}
```

### مفاتيح التخزين في localStorage:
- `wancharisi_demo_uploaded_files` - الملفات المحملة
- `wancharisi_demo_extracted_pages` - الصفحات المستخرجة
- `wancharisi_demo_verifications` - تقارير التحقق
- `wancharisi_demo_ai_conversations` - محادثات الذكاء الاصطناعي
- `wancharisi_demo_matching_results` - نتائج المطابقة
- `demo_data_loaded` - علامة التحميل الأولي

## 🔧 إدارة البيانات التجريبية

### حذف جميع البيانات التجريبية:

1. افتح Developer Tools في المتصفح (F12)
2. انتقل إلى تبويب Console
3. انسخ والصق الكود التالي:

```javascript
// حذف جميع البيانات التجريبية
localStorage.removeItem('wancharisi_demo_uploaded_files');
localStorage.removeItem('wancharisi_demo_extracted_pages');
localStorage.removeItem('wancharisi_demo_verifications');
localStorage.removeItem('wancharisi_demo_ai_conversations');
localStorage.removeItem('wancharisi_demo_matching_results');
localStorage.removeItem('demo_data_loaded');

alert('تم حذف جميع البيانات التجريبية');
location.reload();
```

### إعادة تحميل البيانات التجريبية:

```javascript
// إعادة تحميل البيانات
localStorage.removeItem('demo_data_loaded');
location.reload();
```

### عرض البيانات المخزنة:

```javascript
// عرض عدد الملفات المحملة
const files = JSON.parse(localStorage.getItem('wancharisi_demo_uploaded_files') || '[]');
console.log('عدد الملفات:', files.length);

// عرض عدد الصفحات المستخرجة
const pages = JSON.parse(localStorage.getItem('wancharisi_demo_extracted_pages') || '[]');
console.log('عدد عمليات الاستخراج:', pages.length);

// عرض عدد تقارير التحقق
const verifications = JSON.parse(localStorage.getItem('wancharisi_demo_verifications') || '[]');
console.log('عدد تقارير التحقق:', verifications.length);
```

## 📝 ملاحظات مهمة

⚠️ **البيانات التجريبية لا تتعارض مع البيانات الحقيقية:**
- البيانات التجريبية تُخزن بمفاتيح منفصلة (`demo_` prefix)
- يمكن استخدام المنصة بشكل طبيعي مع وجود البيانات التجريبية
- حذف البيانات التجريبية لن يؤثر على بياناتك الحقيقية

✅ **البيانات محفوظة محلياً:**
- جميع البيانات محفوظة في `localStorage` الخاص بالمتصفح
- لا يتم إرسال أي بيانات إلى خوادم خارجية
- خصوصيتك محفوظة بالكامل

🔄 **التحديثات المستقبلية:**
- قد يتم إضافة المزيد من البيانات التجريبية في التحديثات القادمة
- يمكن تخصيص البيانات التجريبية حسب الحاجة
