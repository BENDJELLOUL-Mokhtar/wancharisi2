# 🚀 دليل سريع - نظام التقارير

## 📝 في دقيقتين: كيف تستخدم التقارير؟

---

### 🎯 **الخطوات**

```
1. افتح upload.html
2. اضغط "مراجعة" على أي دراسة
3. مرر إلى "تقارير الخبراء"
4. اختر:
   👤 تقرير بشري → أدخل اسم المراجع
   🤖 تقارير AI → توليد تلقائي (3 تقارير)
5. شاهد التقييمات والتحليلات
6. صدّر أو اطبع (اختياري)
```

---

## 🤖 **التقارير الثلاثة AI**

### 1️⃣ Scholar Agent 🎓
```
التخصص: تحليل أكاديمي
يقيّم: المنهج، البنية، المراجع
يقدم: تحليل تقني كامل
```

### 2️⃣ Quality Agent 🔬
```
التخصص: جودة الملف
يفحص: PDF، التنسيق، الأخطاء
يقدم: فحوصات تقنية شاملة
```

### 3️⃣ Deep Analysis Agent 🧠
```
التخصص: الأثر والقيمة
يحلل: الإسهام، الابتكار
يقدم: تحليل الأثر المستقبلي
```

## 📊 **محتوى كل تقرير**

```
📌 الرأس
   • اسم المراجع/الوكيل
   • التاريخ والوقت

⭐ التقييمات (1-5 نجوم)
   • 4 معايير + معدل إجمالي

✅ نقاط القوة
   • قائمة كاملة

⚠️ نقاط الضعف
   • ملاحظات وتحذيرات

💡 المقترحات
   • توصيات محددة

🔧 التحليل التقني/المتخصص
   • حسب نوع الوكيل

🎯 التوصية النهائية
   • موافقة ✅
   • تعديلات ✏️
   • رفض ❌
```

## 📤 **التصدير**

| الزر | الوظيفة | الاستخدام |
|------|---------|-----------|
| 📄 **PDF** | تصدير كملف PDF | للأرشفة |
| 🖨️ **طباعة** | طباعة مباشرة | للعروض |
| 📊 **ملخص** | تقرير موجز شامل | للقرارات |

---

## 📋 **التقرير الملخص**

### يحتوي على:

```
✅ معلومات الدراسة
✅ إحصائيات المراجعة (عدد التقارير)
✅ التقييمات المجمعة (متوسط كل معيار)
✅ توزيع التوصيات (موافقة/تعديل/رفض)
✅ القرار المقترح (بناءً على الأغلبية)
```

### مثال:

```
═══════════════════════════════════════
📊 إحصائيات المراجعة
═══════════════════════════════════════
عدد التقارير: 4
• بشريين: 1
• AI: 3

⭐ المعدل: 4.3/5

⚖️ التوصيات:
✅ موافقة: 4 (100%)
✏️ تعديل: 0 (0%)
❌ رفض: 0 (0%)

🎯 القرار: ✅ الموافقة
═══════════════════════════════════════
```
