# 🔧 دليل إدارة الدراسات المحملة

## ✨ الميزات الجديدة

تم إضافة إمكانية **التعديل** و **الحذف** للدراسات المرفوعة من خلال نظام الرفع والموافقة عليها.

---

## 🎯 لمن هذه الميزات؟

✅ **الدراسات المرفوعة** عبر نظام الرفع (upload.html)  
✅ **الدراسات المعتمدة** التي تمت الموافقة عليها  
✅ **الدراسات في المكتبة** التي عليها علامة `isUploaded: true`

❌ **لا تظهر** لـ:
- الدراسات التجريبية الأصلية (الموجودة في الكود)

---

## 🔧 ميزة التعديل

### كيف تعدّل دراسة؟

```
1. افتح library.html
2. ابحث عن الدراسة المرفوعة
3. ستجد زر "✏️ تعديل" أزرق اللون
4. اضغط عليه
5. ستفتح نافذة تعديل شاملة
6. عدّل ما تريد
7. اضغط "💾 حفظ التعديلات"
```

### ✅ ما يمكن تعديله:

- ✏️ نوع الدراسة
- ✏️ عنوان الدراسة
- ✏️ اسم المؤلف/الباحث
- ✏️ سنة النشر
- ✏️ عدد الصفحات
- ✏️ الناشر/الجهة
- ✏️ التصنيف
- ✏️ وصف الدراسة
- ✏️ لغة الدراسة
- ✏️ الكلمات المفتاحية
- ✏️ الرابط الإلكتروني

### ❌ ما لا يمكن تعديله:

- 📄 الملف نفسه (PDF)
- 🆔 معرف الدراسة (ID)
- 📅 تاريخ الرفع
- 📊 حالة الموافقة

**ملاحظة**: لتغيير الملف نفسه، يجب حذف الدراسة وإعادة رفعها.

---

## 🗑️ ميزة الحذف

### كيف تحذف دراسة؟

```
1. افتح library.html
2. ابحث عن الدراسة المرفوعة
3. ستجد زر "🗑️ حذف" أحمر اللون
4. اضغط عليه
5. سيظهر تأكيد: "هل أنت متأكد؟"
6. اضغط "موافق" للحذف
7. ستُحذف الدراسة نهائياً
```

### ⚠️ تحذير هام:

```
⚠️ الحذف نهائي ولا يمكن التراجع عنه!
⚠️ سيُحذف الملف والبيانات معاً
⚠️ تأكد قبل الحذف
```

### 📦 ماذا يحدث عند الحذف؟

1. ✅ تُحذف من المكتبة الرقمية (`library_studies`)
2. ✅ تُحذف من الدراسات المرفوعة (`uploaded_studies`)
3. ✅ تُحذف من الصفحة فوراً
4. ✅ تظهر رسالة نجاح

## 🔄 سير العمل

### تعديل دراسة:
```
1. عرض الدراسة في المكتبة
   ↓
2. الضغط على "تعديل"
   ↓
3. ملء النموذج (معبأ مسبقاً)
   ↓
4. تعديل الحقول المطلوبة
   ↓
5. حفظ أو إلغاء
   ↓
6. تحديث فوري في:
   • المكتبة الرقمية
   • الدراسات المرفوعة
   • الصفحة المعروضة
```

### حذف دراسة:
```
1. عرض الدراسة في المكتبة
   ↓
2. الضغط على "حذف"
   ↓
3. تأكيد الحذف
   ↓
4. حذف نهائي من:
   • localStorage
   • الصفحة
   ↓
5. رسالة نجاح
```

## 💾 التخزين

### ما يُحفظ في localStorage:

```javascript
// 1. المكتبة الرقمية
localStorage['wancharisi_library_studies'] = [
    {
        id: 123,
        title: "عنوان محدّث",
        author: "مؤلف محدّث",
        // ... بقية الحقول المحدّثة
        fileData: "base64...", // لا يتغير
        isUploaded: true
    }
]

// 2. الدراسات المرفوعة
localStorage['wancharisi_uploaded_studies'] = [
    {
        id: 123,
        title: "عنوان محدّث",
        // ... يتم التحديث أيضاً
        status: "approved"
    }
]
```

## ⚠️ المشاكل الشائعة

### المشكلة: زر التعديل/الحذف لا يظهر
**الحل**:
- تأكد أن الدراسة مرفوعة عبر نظام الرفع
- تأكد أن `isUploaded: true` في البيانات
- أعد تحميل الصفحة

### المشكلة: التعديلات لا تُحفظ
**الحل**:
- تأكد من ملء جميع الحقول المطلوبة (*)
- تحقق من console (F12) للأخطاء
- تأكد من localStorage غير ممتلئ

### المشكلة: الحذف لا يعمل
**الحل**:
- تأكد من تأكيد الحذف
- تحقق من console للأخطاء
- جرّب مسح localStorage وإعادة المحاولة
