# مجلد الدراسات عن الإمام الونشريسي

## نظرة عامة
هذا المجلد مخصص لتخزين ملفات PDF الخاصة بالدراسات والأبحاث التي تناولت سيرة وتراث الإمام أحمد بن يحيى الونشريسي.

---

## أنواع الملفات المدعومة

### 1. **الكتب الأكاديمية**
كتب علمية ودراسات شاملة عن حياة ومنهج الإمام الونشريسي

### 2. **المقالات العلمية**
أبحاث منشورة في المجلات المحكمة

### 3. **الأطروحات الجامعية**
رسائل الماجستير والدكتوراه

### 4. **أعمال الملتقيات والمؤتمرات**
مجموعات أوراق العمل والأبحاث من المؤتمرات العلمية

---

## كيفية إضافة دراسة جديدة

### طريقتان للإضافة:

#### ✅ الطريقة الأولى: عبر نظام الرفع (موصى به)
1. افتح صفحة [upload.html](../upload.html)
2. انتقل إلى "القسم الثاني: تحميل الدراسات والأبحاث"
3. املأ النموذج بكامل بيانات الدراسة
4. ارفع ملف PDF
5. اضغط على "رفع الدراسة"
6. ستظهر الدراسة في قائمة "الدراسات المرفوعة" بحالة "قيد المراجعة"
7. بعد مراجعة الإدارة، ستُضاف تلقائياً للمكتبة الرقمية

#### ⚙️ الطريقة الثانية: يدوياً (للمطورين)

**الخطوة 1**: رفع ملف PDF
1. ضع ملف PDF للدراسة في هذا المجلد
2. استخدم اسم ملف واضح ومفهوم بالعربية أو الإنجليزية
3. تأكد من جودة الملف وقابليته للقراءة

**مثال على أسماء الملفات الجيدة:**
- `الونشريسي_وجهوده_الفقهية.pdf`
- `منهج_المعيار_المعرب_دراسة_تحليلية.pdf`
- `اطروحة_النوازل_الفقهية_2021.pdf`

### الخطوة 2: تحديث قاعدة البيانات
افتح ملف `library.js` وأضف بيانات الدراسة في مصفوفة `studies`:

```javascript
{
    id: 7, // رقم تسلسلي جديد
    title: 'عنوان الدراسة الكامل',
    author: 'اسم المؤلف أو الباحث',
    type: 'كتاب', // أو: مقال، أطروحة دكتوراه، ملتقى
    year: '2024',
    pages: '300',
    description: 'وصف مفصل للدراسة ومحتواها وأهميتها العلمية',
    publisher: 'جهة النشر أو الجامعة',
    category: 'التصنيف الموضوعي',
    filename: 'اسم_الملف.pdf', // اسم الملف بالضبط كما في المجلد
    hasFile: true // true لأن الملف متوفر
}
```

### الخطوة 3: حفظ وتحديث
1. احفظ التعديلات في `library.js`
2. أعد تحميل الصفحة للتأكد من ظهور الدراسة الجديدة

## المزايا المتاحة

### ✅ البحث الذكي
- البحث في العنوان، المؤلف، الوصف، جهة النشر، والتصنيف
- تمييز النتائج المطابقة بألوان بارزة
- عرض عدد النتائج المطابقة
- إمكانية مسح البحث والعودة للكل

### ✅ الفلترة
- فلترة حسب نوع الدراسة (كتب، مقالات، أطروحات، ملتقيات)
- الجمع بين البحث والفلترة للحصول على نتائج دقيقة

### ✅ التحميل
- تحميل مباشر للدراسات المتوفرة
- إشعارات فورية عند بدء التحميل
- تتبع سجل التحميلات

### ✅ طلب النسخة
- إمكانية طلب الدراسات غير المتوفرة
- إدخال البريد الإلكتروني لإرسال الدراسة عند توفرها
- حفظ الطلبات للمتابعة

## نصائح مهمة

### ⚠️ حقوق النشر
- تأكد من الحصول على إذن بنشر الدراسة إذا لزم الأمر
- احترم حقوق الملكية الفكرية للباحثين
- ذكر المصادر والمراجع بدقة

### 📝 جودة الملفات
- استخدم ملفات PDF عالية الجودة
- تأكد من وضوح النص وقابليته للقراءة
- حجم الملف يجب أن يكون معقولاً (يفضل أقل من 50 ميجابايت)

### 🔍 البيانات الوصفية
- اكتب وصفاً دقيقاً وشاملاً لكل دراسة
- استخدم كلمات مفتاحية مناسبة
- تأكد من صحة معلومات المؤلف وجهة النشر
