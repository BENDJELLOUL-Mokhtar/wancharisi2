## 🚀 كيفية الاستخدام

### الخطوة 1️⃣: البحث في ASJP

```
1. اذهب إلى: https://www.asjp.cerist.dz/
2. اكتب في مربع البحث: "الونشريسي"
3. سيظهر لك قائمة بالمقالات
```

أو استخدم الرابط المباشر:
```
https://www.asjp.cerist.dz/en/Articles/search?q=الونشريسي
```

### الخطوة 2️⃣: نسخ بيانات المقال

عندما تفتح أي مقال في ASJP:

```
✅ انسخ الرابط من شريط العنوان
✅ انسخ عنوان المقال
✅ انسخ اسم المؤلف/الباحث
✅ انسخ اسم المجلة
✅ انسخ سنة النشر
✅ انسخ المجلد والعدد (إن وُجدا)
✅ انسخ الصفحات
✅ انسخ الملخص (إن وُجد)
✅ انسخ الكلمات المفتاحية
```

### الخطوة 3️⃣: إضافة المقال

في صفحة **asjp-articles.html**:

```
1. الصق الرابط في حقل "رابط المقال"
2. الصق باقي البيانات في الحقول المناسبة
3. املأ الحقول المطلوبة (*) على الأقل
4. اضغط "حفظ المقال"
5. ✅ سيظهر المقال في القائمة أدناه
```

### الخطوة 4️⃣: إدارة المقالات

#### عرض المقال في ASJP:
```
اضغط زر 👁️ (أخضر) → يفتح المقال في ASJP
```

#### تعديل المقال:
```
اضغط زر ✏️ (أزرق) → تُملأ الحقول بالبيانات القديمة
→ عدّل ما تريد → احفظ
```

#### تصدير للمكتبة:
```
اضغط زر 📤 (برتقالي) → ينتقل للمكتبة الرقمية فوراً
```

#### حذف المقال:
```
اضغط زر 🗑️ (أحمر) → تأكيد الحذف → ✅ تم
```

### الخطوة 5️⃣: التصدير الجماعي

بعد إضافة عدة مقالات:

```
1. اضغط "تصدير الكل للمكتبة" → جميع المقالات تنتقل للمكتبة
2. أو اضغط "تنزيل JSON" → حفظ نسخة احتياطية
```

## 💾 التخزين

### localStorage:
```javascript
// المفتاح المستخدم
localStorage['wancharisi_asjp_articles']

// البنية:
[
  {
    id: timestamp,
    url: "...",
    title: "...",
    author: "...",
    journal: "...",
    year: "...",
    // ... بقية الحقول
  }
]
```

### النسخ الاحتياطي:
```
1. اضغط "تنزيل JSON"
2. يُحفظ ملف بهذا الاسم:
   wancharisi_asjp_articles_2026-05-03.json
3. احفظه في مكان آمن
```

## 📤 التكامل مع المكتبة الرقمية

### التصدير التلقائي:

عند تصدير مقال، يتم تحويله لصيغة الدراسات:

```javascript
{
  type: 'مقال',
  title: "من ASJP",
  author: "من ASJP",
  year: "من ASJP",
  publisher: journal, // اسم المجلة
  category: 'فقه',
  description: abstract || "مقال منشور في...",
  url: "رابط ASJP",
  hasFile: false,
  isUploaded: true,
  source: 'ASJP' // علامة المصدر
}
```

### المزايا:
- ✅ يحتفظ برابط المقال الأصلي
- ✅ يظهر في المكتبة الرقمية فوراً
- ✅ يمكن تعديله أو حذفه من المكتبة
- ✅ يظهر مع علامة "مصدر: ASJP"
